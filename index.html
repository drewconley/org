My Goals

I know I've had a great day when I was able to contribute to a project, teach something new to an excited student, or learn something new myself. I grew up as a musician, so most of my web projects intend to help musicians in the creative process and promotion. I have the honor of working with many talented young guitar students, so anything I can do to push them forward is a success in my book. I'm also trying to teach my dog how to roll over.

Web Development

Front end technologies such as HTML, CSS, and JavaScript are my speciality. I enjoy engineering tight presentation layers and interactive UI for web applications in the name of good UX. My favorite language to write is JavaScript because of its wide range of use cases, but I also really love Ruby. Angular is my framework of choice for JavaScript applications. Other sweet tools I use every day: Jasmine for unit testing, Gulp for build-task management, Sass for preprocessing CSS. Though, I'm always trying new things :)

Guitar Lessons

Anybody that knows me could tell you that my #1 hobby is playing the guitar. (I talk about it all the time, it's probably really annoying.) I started casually teaching others to play in high school but it slowly grew to a busy side business. The coolest feeling in the world for me is empowering the musically inspired to become real musicians. Learning to write songs was pretty painful for me when growing up, so I try to make it easier for people.
